# DOTween Visual Editor

为 Unity 开发者提供直观、高效的 DOTween 动画可视化编辑体验。

## 核心特性

- 🎯 **组件绑定设计**：数据与物体一对一绑定，降低心智负担
- 🖼️ **可视化编辑器**：UI Toolkit 构建的现代化编辑器窗口
- 🔄 **实时预览**：编辑器内即时预览动画效果
- ⚡ **GC 优化**：对象池机制减少运行时内存分配
- 🔌 **版本兼容**：自动适配 DOTween Free / Pro

## 快速开始

### 安装

通过 Unity Package Manager 从 Git URL 安装：

```
https://github.com/cnoom/DOTweenVisualEditor.git
```

### 前置依赖

- Unity 2021.3 LTS 或更高版本
- DOTween 1.2.765 或更高版本

### 基本使用

1. 在 GameObject 上添加 `DOTweenVisualPlayer` 组件
2. 打开编辑器窗口：`Tools > DOTween Visual Editor`
3. 在编辑器中添加和配置动画步骤
4. 点击预览或运行游戏播放动画
