# DOTween Visual Editor

一个为 Unity 开发者提供的 DOTween 可视化编辑器，让动画制作更直观、更高效。

## 核心特性

- 🎯 **组件绑定**：动画数据直接存储在组件上，无需额外资产文件
- 🖱️ **可视化编辑**：UI Toolkit 构建的现代化编辑器界面
- 📋 **步骤列表**：直观的步骤管理，支持拖拽排序
- ⏱️ **时间轴预览**：可视化时间轴，快速定位动画节点
- 🔄 **实时预览**：编辑器内直接预览动画效果
- 🎨 **DOTween 兼容**：支持 Free 和 Pro 版本

## 快速开始

### 安装

1. 确保项目已安装 [DOTween](https://dotween.demigiant.com/)
2. 通过 Unity Package Manager 从 Git URL 安装：
   ```
   https://github.com/cnoom/DOTweenVisualEditor.git
   ```

### 基本使用

1. 在 GameObject 上添加 `DOTween Visual Player` 组件
2. 右键选择 `DOTween Visual Editor` 打开编辑器窗口
3. 点击 `[+ Add Step]` 添加动画步骤
4. 在步骤面板中配置参数
5. 点击 `[Preview All]` 预览效果

## 支持的动画类型

| 类型 | 说明 |
|------|------|
| Move | 移动动画（绝对/相对） |
| Rotate | 旋转动画（绝对/相对） |
| Scale | 缩放动画（绝对/相对） |
| Property | 通用属性动画（Color、Alpha 等） |
| Delay | 延迟等待 |
| Callback | 回调调用 |

## 执行模式

- **Append**：顺序追加到序列末尾
- **Join**：与上一个 Tween 同时执行
- **Insert**：在指定时间点插入

## 系统要求

- Unity 2021.3 LTS 或更高版本
- DOTween 1.2.0 或更高版本
