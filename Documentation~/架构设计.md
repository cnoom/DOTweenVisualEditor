# 架构设计

## 一、整体架构

```
┌──────────────────────────────────────────────────────┐
│              DOTweenVisualEditorWindow                │
│              (EditorWindow, UI Toolkit)               │
│  ┌──────────────────────────────────────────────────┐ │
│  │  左侧：步骤概览（ListView，拖拽排序）             │ │
│  │  右侧：步骤详情（按类型条件显示字段）              │ │
│  │  工具栏：预览/停止/重播/重置/添加步骤              │ │
│  │  状态栏：播放状态 + 时间显示                      │ │
│  └──────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────┘
            │ 编辑 SerializedProperty
            ▼
┌──────────────────────────────────────────────────────┐
│             DOTweenVisualPlayer                       │
│             (MonoBehaviour，挂在目标物体上)            │
│  ┌──────────────────────────────────────────────────┐ │
│  │  List<TweenStepData> _steps (序列化)              │ │
│  │  Play() / Stop() / Pause() / Resume() / Restart()│ │
│  │  链式回调：OnStart / OnComplete / OnUpdate        │ │
│  └──────────────────────────────────────────────────┘ │
└──────────────────────────────────────────────────────┘
            │ 构建 Sequence
            ▼
┌──────────────────────────────────────────────────────┐
│               TweenFactory (静态工具)                  │
│  CreateTween()   - 根据步骤数据创建对应 Tween          │
│  AppendToSequence() - 配置缓动/延迟/执行模式并加入序列  │
│  ApplyStartValue()  - 应用起始值到目标物体             │
└──────────────────────────────────────────────────────┘
            │ 依赖
            ▼
┌──────────────────────────────────────────────────────┐
│            TweenValueHelper (静态工具)                 │
│  颜色/透明度读写、动画创建                             │

            │ 校验
            ▼
┌──────────────────────────────────────────────────────┐
│          TweenStepRequirement (静态工具)               │
│  Validate() - 校验目标物体是否满足动画类型的组件需求    │
```

## 二、预览系统

```
DOTweenPreviewManager (与 EditorWindow 解耦)
├── SaveSnapshots()   - 预览前保存所有目标的 Transform/颜色/透明度/UI 状态
├── StartPreview()    - 使用 DOTweenEditorPreview 启动编辑器预览
├── PausePreview()    - 暂停
├── ResumePreview()   - 恢复
├── StopPreview()     - 停止（不恢复）
├── Reset()           - 停止 + 恢复快照
└── Replay()          - 恢复快照 + 重新预览
```

预览 Sequence 构建与运行时播放共用 `TweenFactory.AppendToSequence()`，确保编辑器预览和运行时行为一致。

## 三、数据结构

### TweenStepData

```csharp
[Serializable]
public class TweenStepData
{
    // 基本信息
    public bool IsEnabled = true;
    public TweenStepType Type = TweenStepType.Move;

    // 时长控制
    public float Duration = 1f;
    public float Delay = 0f;

    // 缓动
    public Ease Ease = Ease.OutQuad;
    public bool UseCustomCurve = false;
    public AnimationCurve CustomCurve;

    // Transform 值组
    public Transform TargetTransform;         // null 时使用组件所在物体
    public TransformTarget TransformTarget;
    public bool UseStartValue = false;
    public Vector3 StartVector;
    public Vector3 TargetVector;
    public bool IsRelative = false;

    // Color 值组
    public bool UseStartColor = false;
    public Color StartColor;
    public Color TargetColor;

    // Float 值组（透明度/填充量）
    public bool UseStartFloat = false;
    public float StartFloat;
    public float TargetFloat;

    // 特效参数
    public float JumpHeight;
    public int JumpNum;
    public Vector3 Intensity;
    public float ShakeRandomness;
    public int Vibrato;
    public float Elasticity;

    // 执行模式
    public ExecutionMode ExecutionMode = ExecutionMode.Append;
    public float InsertTime = 0f;

    // 回调
    public UnityEvent OnComplete;
}
```

### TweenStepType

```
Move / Rotate / Scale / Color / Fade / AnchorMove / SizeDelta
Jump / Punch / Shake / FillAmount / Delay / Callback
```

### ExecutionMode

| 模式 | 说明 |
|------|------|
| Append | 顺序追加到序列末尾 |
| Join | 与上一个 Tween 同时执行 |
| Insert | 在指定时间点插入 |

### TransformTarget

```
Position / LocalPosition / Rotation / LocalRotation / Scale
PunchPosition / PunchRotation / PunchScale
ShakePosition / ShakeRotation / ShakeScale
```

## 四、Inspector 绘制器

`TweenStepDataDrawer`（CustomPropertyDrawer）根据 `TweenStepType` 条件显示对应值组：

- **Move/Rotate/Scale** - TargetTransform、TransformTarget、UseStartValue/StartVector、TargetVector、IsRelative、同步按钮
- **Color** - TargetTransform、UseStartColor/StartColor、TargetColor、组件校验警告
- **Fade/FillAmount** - TargetTransform、UseStartFloat/StartFloat、TargetFloat、组件校验警告
- **Jump** - TargetTransform、UseStartValue/StartVector、TargetVector、JumpHeight、JumpNum、同步按钮
- **Punch/Shake** - TargetTransform、子类型选择、Intensity、Vibrato、Elasticity（Shake 还有 ShakeRandomness）
- **Delay** - 仅 Duration
- **Callback** - 仅 OnComplete 事件

## 五、DOTween 版本兼容

通过程序集定义（asmdef）的版本定义实现条件编译：

```
com.demigiant.dotween >= 1.2.0 → DOTWEEN
com.demigiant.dotweenpro >= 1.2.0 → DOTWEEN_PRO
```

`TweenValueHelper` 中使用 `#if DOTWEEN_TMP || TMP_PRESENT` 适配 TextMeshPro。
