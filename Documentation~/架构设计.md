## 二、预览系统

```
DOTweenPreviewManager (与 EditorWindow 解耦)
├── SaveSnapshots()   - 预览前保存所有目标的 Transform/颜色/透明度/UI 状态
├── StartPreview()    - 使用 DOTweenEditorPreview 启动编辑器预览
├── PausePreview()    - 暂停
├── ResumePreview()   - 恢复
├── StopPreview()     - 停止 + 恢复快照
├── Reset()           - 停止 + 恢复快照 + 清空快照
├── Replay()          - 恢复快照 + 重新预览
└── Dispose()         - 释放资源
```

预览 Sequence 构建与运行时播放共用 `TweenFactory.AppendToSequence()`，确保编辑器预览和运行时行为一致。

## 三、异步等待

```
TweenAwaitable (CustomYieldInstruction)
├── 只读等待包装器，外部无法修改内部 Tween
├── IsDone / IsCompleted / IsPlaying / IsActive 状态查询
├── keepWaiting - 协程 yield return 支持
├── WaitForCompletion() - 显式等待方法
└── OnDone(Action<bool>) - 完成回调（true=正常完成，false=被终止）

使用方式：
  yield return player.PlayAsync();                    // 协程
  await player.PlayAsync().ToUniTask();               // UniTask
  player.PlayAsync().OnDone(completed => { ... });    // 回调
```

## 四、数据结构

### TweenStepData

```csharp
[Serializable]
public class TweenStepData
{
    // 基本信息
    public bool IsEnabled = true;
    public TweenStepType Type = TweenStepType.Move;

    // 时长控制
    public float Duration = 1f;
    public float Delay = 0f;

    // 缓动
    public Ease Ease = Ease.OutQuad;
    public bool UseCustomCurve = false;
    public AnimationCurve CustomCurve;

    // Transform 值组
    public Transform TargetTransform;         // null 时使用组件所在物体
    public MoveSpace MoveSpace = MoveSpace.World;
    public RotateSpace RotateSpace = RotateSpace.World;
    public PunchTarget PunchTarget = PunchTarget.Position;
    public ShakeTarget ShakeTarget = ShakeTarget.Position;
    public bool UseStartValue = false;
    public Vector3 StartVector;
    public Vector3 TargetVector;
    public bool IsRelative = false;

    // Color 值组
    public bool UseStartColor = false;
    public Color StartColor;
    public Color TargetColor;

    // Float 值组（透明度/填充量）
    public bool UseStartFloat = false;
    public float StartFloat;
    public float TargetFloat;

    // 特效参数
    public float JumpHeight;
    public int JumpNum;
    public Vector3 Intensity;
    public float ShakeRandomness;
    public int Vibrato;
    public float Elasticity;

    // 路径动画值组
    public Vector3[] PathWaypoints;
    public int PathType;
    public int PathMode;
    public int PathResolution;
    public Color PathGizmoColor;

    // 执行模式
    public ExecutionMode ExecutionMode = ExecutionMode.Append;
    public float InsertTime = 0f;

    // 回调
    public UnityEvent OnComplete;
}
```

### TweenStepType

```
// Transform
Move / Rotate / Scale
// 视觉
Color / Fade
// UI
AnchorMove / SizeDelta
// 特效
Jump / Punch / Shake / FillAmount
// 路径
DOPath
// 流程控制
Delay / Callback
```

### ExecutionMode

| 模式 | 说明 |
|------|------|
| Append | 顺序追加到序列末尾 |
| Join | 与上一个 Tween 同时执行 |
| Insert | 在指定时间点插入 |

### MoveSpace

| 空间 | 说明 |
|------|------|
| World | 世界坐标 |
| Local | 本地坐标 |

### RotateSpace

| 空间 | 说明 |
|------|------|
| World | 世界旋转 |
| Local | 本地旋转 |

### PunchTarget / ShakeTarget

| 目标 | 说明 |
|------|------|
| Position | 位置 |
| Rotation | 旋转 |
| Scale | 缩放 |

## 五、Inspector 绘制器

`TweenStepDataDrawer`（CustomPropertyDrawer）根据 `TweenStepType` 条件显示对应值组：

- **Move** - TargetTransform、MoveSpace、UseStartValue/StartVector、TargetVector、IsRelative、同步按钮
- **Rotate** - TargetTransform、RotateSpace、UseStartValue/StartVector、TargetVector、IsRelative、同步按钮
- **Scale** - TargetTransform、UseStartValue/StartVector、TargetVector、IsRelative、同步按钮
- **Color** - TargetTransform、UseStartColor/StartColor、TargetColor、组件校验警告
- **Fade** - TargetTransform、UseStartFloat/StartFloat、TargetFloat、组件校验警告
- **AnchorMove/SizeDelta** - TargetTransform、UseStartValue/StartVector、TargetVector、IsRelative、组件校验警告、同步按钮
- **Jump** - TargetTransform、UseStartValue/StartVector、TargetVector、JumpHeight、JumpNum、同步按钮
- **Punch** - TargetTransform、PunchTarget、Intensity、Vibrato、Elasticity
- **Shake** - TargetTransform、ShakeTarget、Intensity、Vibrato、Elasticity、ShakeRandomness
- **FillAmount** - TargetTransform、UseStartFloat/StartFloat、TargetFloat、组件校验警告
- **DOPath** - TargetTransform、UseStartValue/StartVector、PathWaypoints（路径点编辑）、PathType、PathMode、PathResolution、同步按钮
- **Delay** - 仅 Duration
- **Callback** - 仅 OnComplete 事件

## 六、DOTween 版本兼容

通过程序集定义（asmdef）的版本定义实现条件编译：

```
com.demigiant.dotween >= 1.2.0 → DOTWEEN
com.demigiant.dotweenpro >= 1.2.0 → DOTWEEN_PRO
```
