# DOTween Visual Editor - 重新设计方案

## 一、设计变更原因

**原设计问题**：
- 数据与对象解耦虽理论优雅，但实际复用场景极少（物体位置各不相同）
- ScriptableObject 资产管理增加复杂度
- 用户期望更直接的操作方式

**新设计目标**：
- 数据与组件绑定，降低心智负担
- 可视化编辑器直接编辑组件上的动画
- 界面简洁直观，类似 DOTween Sequence 的可视化操作

### 2.2 数据结构

#### TweenStepData（动画步骤数据）

```csharp
[Serializable]
public class TweenStepData
{
    public string StepName = "New Step";
    public TweenType Type;              // Move/Rotate/Scale/Property/Callback
    public float Duration = 1f;
    public float Delay = 0f;
    public Ease Ease = Ease.OutQuad;
    public AnimationCurve CustomCurve;  // 自定义曲线
    
    // Transform 相关
    public TransformTarget TargetType;  // Position/Rotation/Scale
    public Vector3 TargetValue;
    public bool IsRelative = false;
    
    // 回调相关
    public UnityEvent Callback;
}
```

#### DOTweenVisualPlayer（主组件）

```csharp
public class DOTweenVisualPlayer : MonoBehaviour
{
    [SerializeField] private List<TweenStepData> _steps = new();
    [SerializeField] private bool _playOnStart = false;
    [SerializeField] private int _loops = 1;
    [SerializeField] private LoopType _loopType = LoopType.Restart;
    
    public IReadOnlyList<TweenStepData> Steps => _steps;
    
    public void Play();
    public void Stop();
    public void Pause();
    public void Restart();
}
```

## 四、核心功能

### 4.1 步骤类型

| 类型 | 说明 | 参数 |
|------|------|------|
| **Move** | 移动动画 | 目标位置/相对偏移、时长、缓动 |
| **Rotate** | 旋转动画 | 目标欧拉角/相对偏移 |
| **Scale** | 缩放动画 | 目标缩放值 |
| **Property** | 通用属性 | Color、Alpha、自定义属性名 |
| **Delay** | 延迟等待 | 时长 |
| **Callback** | 回调调用 | UnityEvent |
| **Join** | 并行执行标记 | 与上一 Tween 同时执行 |

### 4.2 执行模式

```csharp
public enum ExecutionMode
{
    Append,   // 顺序追加到序列末尾
    Join,     // 与上一个 Tween 同时执行
    Insert    // 在指定时间点插入
}
```

### 4.3 预览功能

- **单步预览**：点击步骤右侧 [▶] 按钮，预览该步骤效果
- **全序列预览**：点击底部 [Preview All]，在编辑器中播放完整动画
- **时间轴拖拽**：拖动时间轴游标，快速定位到任意时刻

## 五、与原设计对比

| 项目 | 原设计 | 新设计 |
|------|--------|--------|
| **数据存储** | ScriptableObject 独立资产 | MonoBehaviour 组件字段 |
| **对象绑定** | 多对象引用同一资产 | 一对一绑定 |
| **编辑器** | Track/Clip 复杂时间轴 | 步骤列表 + 简易时间轴 |
| **扩展性** | GraphView 节点编辑 | 预置类型 + 自定义类型 |
| **学习成本** | 较高（类似 Timeline） | 较低（类似 DOTween Animation） |
