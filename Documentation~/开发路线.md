## 第一阶段：核心组件重构

### 目标
建立基于 MonoBehaviour 的核心组件架构，替换原 ScriptableObject 设计。

### 任务清单

- [ ] **删除旧数据模型**
  - 删除 `AnimationSequenceAsset.cs`
  - 删除 `TrackData.cs`
  - 删除 `ClipData.cs`
  - 删除 `AnimationStepData.cs`

- [ ] **新建数据结构**
  - `TweenStepData.cs` - 动画步骤数据
  - `TweenType.cs` - 动画类型枚举
  - `ExecutionMode.cs` - 执行模式枚举

- [ ] **新建主组件**
  - `DOTweenVisualPlayer.cs` - 主播放器组件
  - 序列化字段设计
  - Play/Stop/Pause API

- [ ] **保留并优化**
  - `DOTweenAdapter` - 适配器层
  - `TweenPool` - 对象池

### 交付物
- 编译通过的 Runtime 模块
- 可挂载的 DOTweenVisualPlayer 组件
