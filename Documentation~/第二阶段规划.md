# 第二阶段规划：编辑器窗口（UI Toolkit）

## 一、阶段目标

使用 UI Toolkit 构建可视化编辑器窗口，提供直观的动画步骤管理界面。

---

## 二、功能模块

### 2.1 主窗口（DOTweenVisualEditorWindow）

**打开方式**：
- 菜单：`Tools > DOTween Visual Editor`
- Inspector 上的快捷按钮

**窗口布局**：

```
┌─────────────────────────────────────────────────────────┐
│  工具栏：[预览] [停止] [重置] | [添加步骤 ▼] | [设置]    │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │ 步骤列表（可折叠、拖拽排序）                      │   │
│  │ ┌─────────────────────────────────────────────┐ │   │
│  │ │ ▼ 1. Move (Position)           [启用] [删除]│ │   │
│  │ │     Duration: 1.0s | Ease: OutQuad          │ │   │
│  │ └─────────────────────────────────────────────┘ │   │
│  │ ┌─────────────────────────────────────────────┐ │   │
│  │ │ ▶ 2. Rotate (Rotation)         [启用] [删除]│ │   │
│  │ └─────────────────────────────────────────────┘ │   │
│  │ ┌─────────────────────────────────────────────┐ │   │
│  │ │ ▶ 3. Scale                     [启用] [删除]│ │   │
│  │ └─────────────────────────────────────────────┘ │   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
├─────────────────────────────────────────────────────────┤
│  时间轴预览（简易版）                                    │
│  ┌─────────────────────────────────────────────────┐   │
│  │ 0.0s    1.0s    2.0s    3.0s    4.0s            │   │
│  │ |--------|--------|--------|--------|           │   │
│  │ [Move   ][Rotate      ][Scale   ]               │   │
│  │          [Callback]                             │   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

---

### 2.2 步骤列表功能

**基础功能**：
| 功能 | 说明 |
|------|------|
| 显示/隐藏详情 | 点击折叠箭头展开/收起步骤详情 |
| 拖拽排序 | 拖动步骤调整执行顺序 |
| 启用/禁用 | 切换步骤是否参与播放 |
| 删除步骤 | 移除选中的步骤 |
| 复制步骤 | 复制选中的步骤到剪贴板 |
| 粘贴步骤 | 粘贴剪贴板中的步骤 |

**右键菜单**：
- 复制
- 粘贴
- 删除
- 禁用/启用
- 在上方插入
- 在下方插入

---

### 2.3 添加步骤菜单

**下拉菜单内容**：
```
[添加步骤 ▼]
├── Transform
│   ├── Move (Position)
│   ├── Move (LocalPosition)
│   ├── Rotate (Rotation)
│   ├── Rotate (LocalRotation)
│   └── Scale
├── 其他
│   ├── Delay
│   └── Callback
└── Property (未实现)
```

---

### 2.4 工具栏功能

| 按钮 | 功能 | 快捷键 |
|------|------|--------|
| **预览** | 在编辑器中预览动画 | Space |
| **停止** | 停止预览并重置状态 | Escape |
| **重置** | 将物体恢复到预览前状态 | R |
| **添加步骤** | 下拉菜单添加新步骤 | - |
| **设置** | 打开设置面板 | - |

---

### 2.5 设置面板

**可选配置项**：
- 默认 Duration（默认 1.0s）
- 默认 Ease（默认 OutQuad）
- 自动记录预览前状态
- 预览结束后自动重置
- Debug 模式

## 三、技术实现

### 3.1 文件结构

```
Editor/
├── DOTweenVisualEditorWindow.cs    # 主窗口
├── Elements/
│   ├── StepListElement.cs          # 步骤列表元素
│   ├── StepItemElement.cs          # 单个步骤项元素
│   ├── TimelineElement.cs          # 时间轴元素
│   └── ToolbarElement.cs           # 工具栏元素
├── UXML/
│   ├── DOTweenVisualEditorWindow.uxml
│   ├── StepItemElement.uxml
│   └── TimelineElement.uxml
├── USS/
│   ├── DOTweenVisualEditor.uss
│   └── StepItemElement.uss
└── TweenStepDataDrawer.cs          # 已存在，Inspector 绘制器
```

### 3.2 数据绑定

- 使用 `SerializedObject` 绑定 `DOTweenVisualPlayer` 组件
- 使用 `SerializedProperty` 访问 `Steps` 数组
- 使用 `ListView` 实现步骤列表

### 3.3 拖拽排序

- 使用 `ListView.reorderable = true`
- 监听 `itemsAdded` / `itemsRemoved` 事件同步数据

## 五、待确认问题

### 5.1 窗口与 Inspector 的关系

**方案 A**：独立窗口
- 主窗口只显示步骤列表和时间轴
- Inspector 负责步骤详细编辑
- 两者同时显示，互补

**方案 B**：集成窗口
- 主窗口包含步骤列表 + 详情编辑面板
- 不依赖 Inspector

**建议**：方案 A，与 Unity 原生组件（如 Animator）一致

### 5.2 时间轴功能范围

**问题**：第二阶段实现简易时间轴还是完整时间轴？

**简易版**：
- 只显示步骤块位置
- 无法拖拽调整时间

**完整版**：
- 可拖拽调整起始时间
- 可调整 Duration
- 可缩放时间轴

**建议**：第二阶段只做简易版，完整版留到第三阶段

### 5.3 步骤详情显示方式

**方案 A**：折叠显示（参考 DOTween Animation）
- 点击折叠箭头在列表内展开详情
- 复用 `TweenStepDataDrawer` 绘制

**方案 B**：侧边面板
- 选中步骤后在右侧显示详情面板

**建议**：方案 A，更紧凑

### 5.4 多物体编辑支持

**问题**：是否支持同时编辑多个 `DOTweenVisualPlayer`？

**建议**：暂不支持，第二阶段专注单物体编辑
